- **核心引擎复用**：`src/engine` 作为框架无关的纯 TypeScript 渲染内核，被根目录下的 React 应用（`src`）直接引用，并为 Vue 应用（`r-markdown`）提供逻辑移植蓝本，确保 Markdown 解析、数学公式处理及组件渲染逻辑的一致性。
- **多框架并行架构**：采用 Monorepo 风格组织，根目录承载基于 React + Vite 的主编辑器；`r-markdown` 为独立的 Vue 3 + Vite 实现，服务于特定轻量化或移动端场景；`html-anything` 则是基于 Next.js 的全栈工作区，负责更复杂的 AI 代理集成、模板市场及云端部署能力。
- **物理隔离与逻辑共享**：各子应用通过独立的 `package.json` 和构建配置（Vite/Next.js）实现运行时隔离，但共享 `.agent/skills` 中的设计规范与提示词工程，以及 `src/engine` 中沉淀的底层工具函数（如 `markdownParser`、`inlineFormat`），形成“逻辑下沉、表现分层”的协作模式。
