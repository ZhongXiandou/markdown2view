- **入口与状态层**：`App.tsx` 作为根容器，通过 `lib/store.ts` (Zustand + persist) 管理全局状态（内容、主题、模式），实现跨组件数据共享。
- **模式路由层**：`modes/` 目录下包含 `article`、`document`、`card`、`html` 四个独立子模块，采用 `lazy` 动态加载，根据 store 中的 `mode` 字段切换渲染视图。
- **核心渲染引擎**：`engine/` 模块提供框架无关的纯 TS 解析能力。`markdownParser.ts` 将 Markdown 转换为带内联样式的 HTML，支持自定义标签（如 `<title>`、`<steps>`）映射到 `editor-components/` 中的设计原子（DA01/DA02）。
- **同步与交互层**：`lib/useEditorDocSync.ts` 封装了编辑器与 Store 的双向防抖同步逻辑，解决回声问题；`lib/render/` 和 `lib/export/` 负责最终的 DOM 渲染与 PNG/PDF/ZIP 导出。
- **UI 组件层**：`components/` 提供通用的编辑器工具栏、预览沙箱及 UI 基础控件，通过 Tailwind CSS 实现响应式布局。
