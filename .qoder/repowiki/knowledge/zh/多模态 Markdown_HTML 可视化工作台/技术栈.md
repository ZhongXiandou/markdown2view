- **核心框架**：React 18 + TypeScript + Vite。
- **状态管理**：Zustand (含 `persist` 中间件)。
- **编辑器内核**：CodeMirror 6 (`@codemirror/lang-markdown`, `@codemirror/lang-html`)。
- **导出工具**：`jspdf` (PDF 生成), `modern-screenshot` (DOM 转图片), `html2canvas` (备选方案)。
- **样式方案**：Tailwind CSS (原子类布局) + 动态 CSS Variables (主题色注入)。
