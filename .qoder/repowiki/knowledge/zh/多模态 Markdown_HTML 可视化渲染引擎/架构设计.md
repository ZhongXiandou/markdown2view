- **入口与路由**：`App.tsx` 作为根组件，通过 `ModeTabs` 切换四种渲染模式（Article, Document, Card, Html），并采用 `React.lazy` 实现各模式工作台的按需加载。
- **状态管理**：基于 `zustand` 构建全局 Store (`lib/store.ts`)，集成 `persist` 中间件实现本地持久化；内置版本控制机制（`DEMO_VERSION`）与脏检查（Dirty Flag），确保示例内容更新时不覆盖用户手动编辑的数据。
- **渲染引擎层**：`engine/` 目录包含框架无关的纯 TypeScript 解析器。`markdownParser.ts` 将自定义 Markdown 语法（如 `<title>`, `::: cta`）转换为内联样式的 HTML；`editor-components/` 采用插件化设计，通过 `ComponentDef` 接口统一管理各类 UI 组件的渲染逻辑与元数据。
- **同步与交互**：`lib/useEditorDocSync.ts` 实现了编辑器本地状态与全局 Store 的双向防抖同步，利用 `externalVersion` 信号解决外部重置（如恢复示例）与内部输入之间的竞态问题。
- **沙箱预览**：`HtmlMode` 利用 `iframe` 构建隔离的沙箱环境渲染 HTML，并通过 `ResizeObserver` 与 `MutationObserver` 实现内容的自适应缩放与稳定性探测，确保导出截图的像素级还原。
