- **核心框架**：React 18 + TypeScript + Vite。
- **编辑器内核**：CodeMirror 6 (`@uiw/react-codemirror`)，支持 Markdown 与 HTML 语言扩展及自定义快捷键。
- **状态管理**：Zustand 配合 `zustand/middleware` 实现持久化存储。
- **截图导出**：`modern-screenshot` 用于 DOM 到 Canvas/Blob 的高保真转换，支持 iframe 内复杂布局的精准捕获。
- **样式方案**：Tailwind CSS 用于应用层 UI 布局，渲染产物采用内联 CSS 以确保跨平台兼容性。
