# 自由画布 PPT 导出实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为自由画布（HTML）模式新增两种 PPT 导出能力——图片型（高保真、不可编辑）与可编辑型（实验性、文本可编辑）。

**Architecture:** 复用现有 `captureElementInIframeToBlob` 逐页截图管线。图片型 PPT 将每页截图作为全屏背景贴入 `pptxgenjs` 幻灯片；可编辑型 PPT 解析 iframe DOM 提取文本/图片/基础样式映射为 PPTX 原生元素。两者均通过动态 `import('pptxgenjs')` 按需加载。

**Tech Stack:** React 18 + TypeScript + pptxgenjs（新增）+ Vitest

## 关键设计决策

1. **图片型 PPT 按钮**：`variant: 'primary'`（强调按钮），与 PDF 并列。
2. **可编辑 PPT 按钮**：`variant: 'ghost'`，tooltip 明确标注"实验性功能，复杂样式会丢失"。
3. **幻灯片布局**：根据首页截图比例 `defineLayout` 自定义尺寸（宽 10 英寸，高按比例），所有页共享同一布局。
4. **图片型**：`scale: 3, type: 'image/jpeg'`，与 PDF 导出一致。
5. **可编辑型**：仅提取叶子文本容器（无子元素的文本节点）+ `<img>` 元素；字体大小按 `px * scaleX * 72` 换算为磅；颜色转 6 位 HEX。
6. **可编辑型适用范围**：多页与单页均支持，单页时以 `body > div/main/section` 包裹器为唯一幻灯片。
